#A-La-Carte
<p>A-la-carte is a sails web application that allows users to find food stops en route between two locations during a road trip. It uses the Google Maps API along with the Yelp API to process searches.</p>

<p>To run application</p>
<p>1) Run "npm install -g sails"</p>
<p>2) Run "sails lift"</p>
<p>3) Go to localhost:1337</p>

<p>You can also visit the webapp at http://www.ala-carte.org</p>
