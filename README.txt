Setup:

1) Install node.js onto computer
2) Run "sudo npm -g install sails"
3) Run "sudo npm install --save oauth-signature"
4) Run "sudo npm install --save nonce"
5) Run "sudo npm install --save request"
6) Run "npm install merge --save"
7) Run "npm install node-yelp-api --save"

Starting the app:

1) Run "sails lift"
2) Go to localhost:1337
